# FRC COTS Vendor Quick Reference

## Thrifty Bot (thethriftybot.com)
- Thrifty Swerve (budget swerve modules)
- Thrifty Elevator Stage Kit (~$60/stage, bearing blocks + chain studs + spring mounts)
- Thrifty Nova motor controller
- Thrifty Cycloidal gearbox (designed with 118 Robonauts)
- Budget structural, hardware, sensors

## WCP / West Coast Products (wcproducts.com)
- Kraken X60 motor ($238 each)
- GreyT Cascade Elevator v4
- GreyT Telescope (1-4 stages, for climbers)
- GreyT Turret
- GreyT Shooter (adjustable hood)
- Competitive Concept (CC) — full reference robot released on kickoff day
- Structural tube (plain + punched patterns)
- WCP Friction Brake, Energy Chain

## REV Robotics (revrobotics.com)
- NEO Brushless / NEO Vortex motors
- SPARK MAX / SPARK Flex motor controllers
- MAXSwerve modules
- MAXTube (1x1, 2x1 with hole patterns)
- MAXLift Bearing Blocks (elevator components)
- MAXPlanetary gearbox system
- REV ION FRC Starter Bot — reference robot released on kickoff day

## AndyMark (andymark.com)
- SDS MK4i / MK4n / MK5 swerve modules
- AM14U chassis
- Compliant wheels, flex wheels, stealth wheels
- Pneumatics components
- Field elements for practice
- KitBot components

## Limelight (limelightvision.io)
- Limelight 4 (recommended, includes Hailo accelerator for neural detection)
- Limelight 3 (budget option)

## Strategy: Buy from the vendor with the best price + fastest shipping for each component. Don't be loyal to one vendor.
