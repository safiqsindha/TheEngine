# FRC Scouting Tool Directory

## Data Collection Tools
- Maneuver (frc-maneuver.com) — Offline-first scouting suite, 2026 configured
- ScoutingPASS (PWNAGERobotics GitHub) — JSON-configurable QR code scouting
- SPOT (team3061.org/spot) — Modular, no-code config, 2026 preconfigured
- Lovat (lovat.app) — Shared database scouting platform
- Open Scouting (Team 3484 GitHub) — Web-based, TBA integration
- SPAMalytics (Team 180) — Google Sheets, "Everybot of scouting"
- ObsidianScout — Rebuilt scouting app for 2026
- ElectronScout — Simple, entertaining scouting app
- 1678 Citrus Circuits Suite — 7-app scouting system (gold standard)

## Analysis Tools
- Statbotics (statbotics.io) — EPA ratings, Python API, REST API
- ArcBotics (Chrome extension) — Overlays EPA on TBA pages
- Quick Pick (quick-pick-psi.vercel.app) — Alliance pick list from Statbotics
- Blue Banner Engine — AI scouting analysis platform
- Orpheus — Tool for analyzing scouting data
- FRC Score Optimizer — Identifies highest-impact scoring improvements
- GoatSim — Shoot-on-move fuel collision visualizer
